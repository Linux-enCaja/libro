\section{Recursos}

Para la realización del curso es necesaria la siguiente infraestructura:

\begin{itemize}
  \item Un PC por cada dos asistentes, con las siguientes especificaciones mínimas (\textbf{PODRÍA REVISAR QUE ESTAS ESPECIFICACIONES ESTÉN ACTUALIZADAS? EN ESPECIAL LOS 2GB LIBRES EN DISCO ME PARECEN MUY POCO.}):
    \begin{itemize}
      \item Sistema operativo linux (Ubuntu preferiblemente).
      \item Memoria RAM 512MB.
      \item Procesador de 800 MHz.
      \item Espacio libre en disco 2GB.
      \item Puerto serie.
    \end{itemize}
  \item Tarjeta de Desarrollo AndroidStamp (incluida dentro del costo del curso).
\end{itemize}

\section{Requisitos}
El curso esta dirigido a profesionales en las áreas de Electrónica Digital y Ciencias de la Computación, o cualquier persona con conocimiento básico de Sistemas Digitales, programación en C y/o C++ y arquitectura de computadores.

\section{Material didáctico}
A los asistentes al curso se les suministrará el siguiente material:
\begin{itemize}
  \item Plataforma de desarrollo AndroidStamp.
  \item Acceso a todas las presentaciones.
  \item Software de desarrollo de libre distribución.
  \item Código fuente necesario para realizar las prácticas.
\end{itemize}
